## 📖 Project Description

The **Farm Management System (FMS)** replaces manual farming records with centralized data storage and automated validation. By leveraging the **Salesforce cloud infrastructure**, it provides real-time insights through interactive dashboards and reports. This ensures accurate, secure, and timely agricultural data management for farmers, farm managers, and system administrators.

<details>
<summary><b>🎯 Core Objectives <i>(Click to expand)</i></b></summary>
<br>

- **Digitize Operations:** Migrate from paper registers to a digital platform.
- **Improve Accuracy:** Validate data entry automatically to prevent mistakes.
- **Enable Real-time Monitoring:** Provide instant reports on farm productivity and sales.
- **Ensure Security:** Implement robust role-based access controls to protect sensitive data.
- **Enhance Efficiency:** Streamline record handling and foster better decision-making capabilities.

## 📂 Project Structure

The underlying structure relies on standard Salesforce structural configuration and metadata:

- **Presentation Layer**: Salesforce Lightning App (`Farm Management App`) & Custom Tabs.
- **Data Layer**: Custom Objects (e.g., `Farm Record`), Custom Fields (Text, Phone, Picklist, Number).
- **Security & Access**: Profiles, Roles, and Sharing Rules.
- **Analytics Layer**: Real-time Reports and Dashboards.

## 🚀 Installation / Setup

To deploy this project to your Salesforce Org, follow these steps:

1. Create a **Salesforce Developer Org** (sign up at [developer.salesforce.com/signup](https://developer.salesforce.com/signup)).
2. Log in via `login.salesforce.com`.
3. Navigate to **Setup** > **Object Manager** to create the required Custom Objects (e.g., `Farm Record`).
4. Create necessary Custom Fields (`Farmer Name`, `Contact Number`, `Crop Type`, `Farm Size`) and configure Relationships.
5. Set up **Custom Tabs** for the respective objects.
6. Navigate to the **App Manager** and create a **New Lightning App** named `Farm Management App`.
7. Add the Custom Objects, Reports, and Dashboards as your primary Navigation Items.
8. Configure **Profiles & Roles** to ensure proper access control based on user requirements.

## 💻 Usage

1. Log in to your Salesforce Org.
2. Click the **App Launcher** (nine-dot icon) and search for the **Farm Management App**.
3. Use the relevant data tabs (like **Farm Record**) to create new records. Fill in the required fields correctly (e.g., Farm Name: _Green Farm_, Crop Type: _Rice_).
4. Save the records—validation rules will ensure data integrity automatically.
5. Navigate to the **Reports** and **Dashboards** tabs to analyze system statistics, farm activities, and crop availability.
